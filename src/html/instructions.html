Инструкция по применению гибрида красного калифорнийского червя

Гибрид красного калифорнийского червя (далее червь) используется для переработки органических отходов и получение из этих отходов органического удобрения – биогумус. Данная инструкция разработана на применение 20 000 шт. червей всех возрастов (т.е. один ящик).

Инструкция по применению жидкого биогумуса

Жидкое натуральное универсальное удобрение на основе биогумуса применяется для корневых и внекорневых подкормок любых цветов, комнатных и декоративных растений, цветочной рассады, а также для замачивания семян, клубней и луковиц цветов и укоренения черенков.

Инструкция по применению вермичая

«Вермичай» обеспечивает безопасную, естественную и полностью органическую окружающую среду для растений и почвы. Обработка растений «вермичаем» начинается с предпосевной обработки семян, луковиц, клубней, черенков, корневой части растений.

Инструкция по применению Биогумуса

Биогумус – новое качественное органическое комплексное удобрение, продукт переработки органических отходов популяцией дождевого червя «Самарский гибрид». Представляет собой сыпучую мелкогранулированную массу темно-коричневого цвета.
